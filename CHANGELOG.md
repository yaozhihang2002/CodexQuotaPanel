# Changelog

## v0.2.0 — Pre-release

- 详情面板改为统一的逻辑坐标与 DPI 布局，高缩放屏幕下窗口、控件和卡片内部绘制保持同一比例。
- 修复 125%—175% 缩放时面板底部出现大块空白、文字重叠和进度条错位的问题。
- 悬浮球拖动更新提高到约 120 FPS，在高刷屏上保持顺滑，同时合并高采样率鼠标的冗余事件。
- 拖动期间暂停火焰动画计时器，松开后恢复，降低透明悬浮窗的合成与重绘负担。
- 保留 v0.1.1 的位置持久化、首次显示恢复和中英双语安装改进。

### 下载建议

- 普通用户：`CodexQuotaPanel-0.2.0-Setup.exe`
- 便携使用：`CodexQuotaPanel-0.2.0-portable-x64.zip`
- 仅需中文 MSI：`CodexQuotaPanel-0.2.0-x64.msi`

## v0.1.1 — Pre-release

- 点击“保存并应用”后立即持久化设置，但不再关闭设置窗口。
- 保存后可以继续修改；取消或关闭只撤销上一次保存之后的未保存内容。
- 保存状态改为“更改已保存 · 可继续修改”，反馈更加明确。
- README 增加醒目的安装包与发布页面直达链接。
- 安装向导的“快捷方式”“安装目录”和“确认安装”页面现在可以正常双向返回。
- 欢迎页移除强硬的版权警告，改为简洁的 MIT 开源许可说明。
- 修复安装目录页可能出现的 Windows Installer 2707 错误。
- 再次运行同版本 Setup 时自动执行就地覆盖，无需先手动卸载。
- 退出程序时始终保存可恢复的悬浮球位置，修复重新打开后位置重置的问题。
- 设置同时写入本地 JSON 文件并兼容原注册表，避免注册表账户或权限差异导致位置保存失效。
- 修复窗体首次显示事件再次执行默认定位、导致悬浮球短暂出现在保存位置后跳回右下角的问题。

### 下载建议

- 普通用户：`CodexQuotaPanel-0.1.1-Setup.exe`
- 便携使用：`CodexQuotaPanel-0.1.1-portable-x64.zip`
- 仅需中文 MSI：`CodexQuotaPanel-0.1.1-x64.msi`

## v0.1.0 — Pre-release

CodexQuotaPanel 的首个公开测试版本。

### 主要内容

- 提供五小时与一周额度双环、动态托盘图标和三种火焰反馈样式。
- 支持深色、浅色、跟随系统主题，以及简体中文与英文界面。
- 支持置顶、透明度、鼠标穿透、自由拖动、悬浮球尺寸和字体大小调节。
- 改进悬浮球与详情面板的展开/收起过渡，减少闪烁、残影和方形中间帧。
- 提供额度提醒、免打扰时段、本地趋势与重置卡到期信息。
- 安装前可选择语言，默认简体中文；可选择安装目录和是否创建桌面快捷方式。
- 修复桌面快捷方式不显示应用 Logo 的问题。
- “数据与关于”页面加入 GitHub 项目入口和版本更新说明。

### 下载建议

- 普通用户：`CodexQuotaPanel-0.1.0-Setup.exe`
- 便携使用：`CodexQuotaPanel-0.1.0-portable-x64.zip`
- 仅需中文 MSI：`CodexQuotaPanel-0.1.0-x64.msi`

这是预发布版本，仍可能存在兼容性或界面问题。欢迎提交 Issue、Pull Request 和二创版本。
